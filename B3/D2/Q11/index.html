CSS Box Model Demonstration

Box 1 — Content Box

box-sizing: content-box
Padding and border are added to the width.
Useful for precision spacing, but can cause layout overflow.

Box 2 — Border Box

box-sizing: border-box
Padding and border are included in the width.
Great for layout consistency and responsive design.

Box 3 — Custom Layout

Uses content-box again with different padding, margin, and border.
Highlights how box model properties can be mixed for unique designs.
